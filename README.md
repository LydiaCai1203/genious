# Genious

## 1. 背景

```markdown
计算机及相关专业招生人数的增多，未来几年计算机行业的人会越来越多。
应届毕业生、1～3 年的打工人，往往 0 项目经验 或是 边缘项目经验，因此可能会有需要虚构项目经验的需求出现。
```

## 2. 核心功能

```markdown
基于 AI，根据 目标行业的开源项目技术文档、招聘要求中所需要使用的技术栈，新生成一个项目，或是在旧项目中虚构新功能点，使之能使用到企业所需要的技术栈。用户可以选择简历模板。结合生成一个新的精美的简历，增加面试的命中率。
```

## 3. 输入输出

```markdown
+ 输入
旧简历(pdf、txt)
期望薪资
目标岗位
期望城市（精度：市级别）
目标行业(可选)

+ 输出
新简历(pdf)
```

## 4. 功能模块

```markdown
1. 数据准备
招聘要求 (boss)
开源项目的技术文档、代码
简历模版
示范简历（Java、Golang、Python)

2. 旧简历解析
pdf 解析出 (姓名、年龄、头像、技术栈、项目细节)

3. 检索
根据期望薪资、目标岗位、期望城市，目标行业(没填则选最近所在公司所在行业) 检索出 招聘要求 && 目标行业(没填则选最近所在公司所在行业)内能找到的开源项目

4. LLM 合成新简历
全新的项目经验 或 在旧的项目经验上虚构新的功能点
```
